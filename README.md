# Lantern-Charitable-Trust-Website-Design
End-to-end NGO website design project involving UI/UX design, prototyping, and website development using Figma and Framer.

🌟 **Project Overview**

This project was created for a charitable organization to establish a professional and engaging online presence. The website was designed to effectively communicate the organization's mission, programs, achievements, and community impact while providing visitors with easy access to important information.

The project involved the complete design and development process, from creating the user interface in Figma to building the final website using Framer.

🎯 **Objectives**

- Create a modern and professional website for the organization.
- Improve the organization's online visibility and credibility.
- Present information in a clear and user-friendly manner.
- Ensure accessibility across different devices and screen sizes.
- Provide an engaging experience for visitors, donors, and volunteers.

🛠️ **Tools Used**

- Figma – UI/UX Design and Prototyping
- Framer – Website Development and Deployment

✨ **Features**
- Responsive design for desktop and mobile devices
- Clean and intuitive navigation
- Home section introducing the organization
- About section highlighting the mission and vision
- Programs section showcasing organizational initiatives
- Gallery section displaying activities and events
- Contact section for communication and inquiries
- Professional layout focused on trust and accessibility

📋 **Website Structure**

The website consists of the following sections:

- **Home** – Introduction to the organization and its mission.
- **About** – Information about the organization's background, vision, and objectives.
- **Programs** – Details of initiatives, activities, and services offered.
- **Gallery** – Showcase of events, activities, and community engagement.
- **Contact** – Contact information and communication details for visitors.
